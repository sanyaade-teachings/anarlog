## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-save`
- `allow-delete-secret`
- `allow-get-bool`
- `allow-get-number`
- `allow-get-secret`
- `allow-get-str`
- `allow-repair-keychain-access`
- `allow-set-bool`
- `allow-set-number`
- `allow-set-secret`
- `allow-set-str`
