## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-execute`
- `allow-execute-proxy`
- `allow-execute-transaction`
- `allow-list-meetings`
- `allow-get-meeting`
- `allow-get-meeting-transcript`
- `allow-get-recurring-meeting-history`
- `allow-get-legacy-cleanup-status`
- `allow-cleanup-legacy-files`
- `allow-get-e2ee-identity-status`
- `allow-inspect-e2ee-recovery-key`
- `allow-create-e2ee-identity`
- `allow-import-e2ee-identity`
- `allow-subscribe`
- `allow-unsubscribe`
- `allow-bind-cloudsync-account`
- `allow-configure-cloudsync-token`
- `allow-stop-cloudsync`
- `allow-suspend-cloudsync`
- `allow-get-cloudsync-status`
