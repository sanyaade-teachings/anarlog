## Recording

- Scheduled meeting notes now open before meetings without starting the mic too early, then can start automatically when the meeting begins
- Auto-stop is more reliable after meetings end, including recordings started from detected meeting apps
- Echo cancellation now better aligns speaker audio with microphone input to reduce feedback and duplicate speech
- Active meeting notes should lag less while transcripts and notes are saved in the background

## Transcription

- Soniqo on-device transcription is more stable across realtime and batch modes, with safer model and language handling
- Live transcription now falls back to a supported language when a provider cannot handle every selected spoken language
- Resumed live sessions keep earlier transcript history visible while new words stream in
- GPT and Soniox batch transcripts now save complete text and word timing more consistently

## Notes & AI

- Summary generation handles overloaded providers better and retries with clearer failure handling
- Empty summaries are generated automatically after batch transcription so completed sessions do not get stuck blank
- Template creation is reachable from an empty templates view
- Template-based enhancements fall back gracefully if the selected template cannot be loaded

## UI & Account

- Pro trial start and end dialogs now explain plan changes more clearly
- Settings navigation is cleaner, with Data moved into its own tab and language selection simplified
- Chat controls now live inside the chat panel, and the assistant is labeled Anarlog AI
- App icon and installer artwork updated for the Anarlog brand
- Notification footer actions such as Ignore no longer open the main window
