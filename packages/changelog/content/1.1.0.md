date: "2026-06-24"
summary: "Anarlog now adds Cartesia transcription, smarter chat context, better meeting controls, and a steadier editing experience."
---

## Transcription

- Cartesia is now available as a speech-to-text provider with Ink 2 live transcription and batch fallback support
- Transcript speaker labels can now be reassigned from the transcript, either across matching speaker turns or for a single segment
- Speaker corrections now carry through transcript rendering and exports more reliably
- Live captions now use a native desktop surface for a cleaner, more reliable viewing experience

## Notes and Summaries

- Summary titles now generate before note content so new AI notes feel more complete as they appear
- Short transcripts now still show the Summary tab with an empty summary note when auto-enhance is skipped
- Active sessions and recordings are preserved more safely when empty tabs close or retention cleanup runs
- Collapsed session headers now keep a stable height as notes load or close
- Long session titles now keep the surrounding header area draggable instead of stretching the title hitbox across the window
- Note tabs now use clearer icons, and empty notes now show the memo tab so there is always an obvious place to start writing
- Clicking editor whitespace now creates a new blank line, making it easier to continue writing at the end of a note
- Standalone note windows now open more predictably, keep title edits hydrated, and avoid title-editor crashes on empty memos

## Chat

- Chat now keeps follow-up requests queued so follow-up work can continue more smoothly
- Chat context now appears as input chips, making it easier to see what the assistant is using
- Floating chat controls now resize and expand more predictably, with a cleaner composer and modal shape

## Meetings and Notifications

- Cal.com meetings now include a join control directly in the meeting surface
- Collapsed sidebars no longer show the upcoming meeting badge when that meeting note is already open
- The left sidebar is now resizable and starts at a more comfortable default width
- Notification close buttons now stay inside the notification panel and keep a round shape while hovering

## Settings

- The Intelligence provider picker no longer shows a Pro chip next to gated providers while preserving upgrade prompts for locked choices
- Template browsing now uses a simpler list with tighter spacing and more consistent header styling
