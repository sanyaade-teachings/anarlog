## Performance

- Typing in long notes now stays smoother, especially while live transcript updates are arriving
- Starting a recording now causes less sidebar and editor movement, so the note surface settles faster
- Meeting timelines, note rows, and action menus now refresh less often while the app is busy
- Collapsed timeline panels now stay idle until you open them

## Notes

- Note edits now avoid applying the same editor update twice, keeping active notes steadier
- Moving between notes and meeting surfaces now reuses more of the existing view state
- Background transcript keyword work now waits until it is needed, reducing startup load

## Shortcuts

- Removed older tab shortcut controls that no longer matched the current note layout
- Main note shortcuts now focus on navigation and app actions that still apply to the current workspace

## Localization

- Localized builds now keep translation catalogs more consistent between releases
