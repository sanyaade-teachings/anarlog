summary: "Anarlog makes large note libraries faster, adds flexible summary instructions, improves recording recovery, and strengthens CloudSync reliability."
---

## Performance

- Notes open faster from calendar events, even with a large library
- Search indexing now runs incrementally in the background so typing and navigation stay responsive

## Summaries

- Custom summary instructions now support template variables and show their estimated token usage
- AI setup appears only when a summary actually needs provider configuration
- Completed summaries refresh and persist more reliably

## Developers

- Install the bundled Anarlog CLI from Settings, then browse local meetings from scripts and coding agents with the [CLI guide](https://docs.anarlog.so/agents/cli)
- Connect agents to notes, summaries, and transcripts using the read-only local server in the [MCP guide](https://docs.anarlog.so/agents/mcp)

## Recording recovery

- Incomplete recordings can be resumed or retranscribed directly from the note

## Reliability

- Pro CloudSync stays bound to the correct account and workspace
- CloudSync now recovers more safely from failures and shuts down cleanly with the app database
- CloudSync and live note updates now coexist safely across database transactions
- Secure API key storage keeps a consistent identity across app updates

## Polish

- Transcript controls remain visible while expanding the floating bar
- macOS window controls align consistently across development and release builds
- Onboarding demo controls and support links are clearer and more reliable
- Storage, microphone, permissions, dictionary, AI, and summary settings are more compact and consistent
